## ğŸ› ï¸ Troubleshooting

### Prompt Install Tidak Muncul?
1. **Pastikan** Anda menggunakan HTTPS (secure connection)
2. **Coba** tutup dan buka kembali browser
3. **Gunakan** metode install manual (lihat panduan di atas)
4. **Clear** cache browser dan coba lagi

### Di iOS Tidak Bisa Install?
1. **Pastikan** menggunakan browser **Safari** (bukan Chrome/Firefox)
2. **Update** iOS ke versi terbaru
3. **Cek** setting Safari: Settings > Safari > Advanced > JavaScript (harus ON)

### Aplikasi Tidak Update?
1. **Tutup** aplikasi sepenuhnya (force close)
2. **Buka** kembali aplikasi
3. **Tunggu** beberapa detik, aplikasi akan auto-update
4. Atau **uninstall dan install ulang** untuk update manual

### Aplikasi Lambat?
1. **Clear** cache aplikasi:
   - Android: Settings > Apps > FinaryApp > Storage > Clear Cache
   - iOS: Uninstall dan install ulang
2. **Restart** perangkat
3. **Pastikan** storage perangkat tidak penuh
